Defaults

定義された各々の SchemaType (これらについてより詳しく知りたければ model definition の章を参照ください) は、 default 値を持ちます。

document スケルトンが構築されたときに default 値は、適用されます。 つまり、新規に documentを生成したとき (new MyModel) 、ないしは、既存の document を検索したとき (MyModel.findById など) を意味し、また、key がない場合にも提供されます。

Definition

関数を伴った定義を行う

new Schema({
    date: { type: Date, default: Date.now }
})

または、値で指定する

new Schema({
    date: { type: Date, default: '12/10/1990' }
})

default は自動的にキャストされることを意識してください。 default は、 Date オブジェクトになります。 どちらの場合でも、default が実際の Date になり、しかし、我々は、最初に timestamp で与えてしまうと、 文字列の date 秒を渡します。
