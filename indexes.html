Indexes

index は、指定した connection / database に対して、モデルがコンパイルされるごとに、 ensureIndex を利用して定義されます。 アプリのライフタイムのあいだに一度、 index が確保されることを意味します。

Definition

通常の index

var User = new Schema({
    name: { type: String, index: true }
})

ユニーク index

var User = new Schema({
    name: { type: String, unique: true }
})
 
// or 
 
var User = new Schema({
    name: { type: String, index: { unique: true } }
})

Schema で定義される複合 index

User.index({ first: 1, last: -1 })
